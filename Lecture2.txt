Primitive data types

int - integer = signed 32 bit (0 to 2147483647)
long - long integer = signed 64-bit (0 to 9223372036854775807)
float   - floating point number = signed 32 bit (0 to 3.4028235E38)
double  - double precision floating point number = signed 64 bit (0 to 1.7976931348623157E308)
boolean - boolean = signed 8 bit (true or false)
char    - character = signed 16 bit (0 to 65535)
byte    - byte = signed 8 bit (0 to 255)
short   - short integer  = signed 16 bit (0 to 65535)
